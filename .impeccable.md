## Design Context

### Users
School staff and students at SMK Penanti, Kubang Semang, Pulau Pinang. Five roles:
- **Admin** — school administrators managing users, classes, exams
- **Teacher** — class teachers and subject teachers entering marks, managing OMR
- **Subject Coordinator** — panitia approving marks and answer schemes
- **Principal** — pengetua viewing dashboards and class-teacher assignments
- **Student** — pelajar viewing their own results and report cards

Used during school hours on desktop/laptop, occasionally on tablet. Malay-language UI.

### Brand Personality
Authoritative · Precise · Dignified

### Aesthetic Direction
Editorial dark — pure black background, vivid blue primary, white text. Inspired by premium institutional publications. High contrast, typographic authority. Right panel always uses `bg-card` (dark charcoal) for visual separation.

**Anti-references**: Generic SaaS dashboard, colorful admin panels, glassmorphism, icon-box-per-feature patterns.

### Fonts
- **Display**: Bodoni Moda (headings, labels, emphasis) — loaded via Next.js font, variable `--font-display`
- **Body**: Figtree (all body text, UI labels) — loaded via Next.js font, variable `--font-sans`

### Color Tokens (dark mode — client-approved)
- `--background`: `oklch(0 0 0)` — pure black
- `--card`: `oklch(0.1684 0 0)` — dark charcoal, elevated surfaces
- `--primary`: `oklch(0.5683 0.2366 264.2478)` — vivid blue
- `--border`: `oklch(0.2746 0.0130 279.0717)` — very subtle

### Design Principles
1. **Typography does the work** — headings are big and editorial, body text is tight and precise. No icon-box grids.
2. **Two surfaces only** — black (`bg-background`) and charcoal (`bg-card`). Elevation = `bg-card`, not shadows or borders.
3. **Blue is rare** — primary blue only on CTAs, active states, and one emphasis word per heading. Never for decorative fills.
4. **Numbers over icons** — numbered lists (`01 / 02 / 03`) replace icon-card grids everywhere.
5. **No `* { font-size: 14px }` fight** — use `!text-[Xpx]` with `style={{ fontSize: 'inherit' }}` on child `em` elements for display headings.

### CSS Layout Patterns
- Login split: `data-layout="login-split"` (defined in globals.css) → `1fr 480px` at lg+
- Full-bleed: no `max-w` on the outer grid, surfaces fill viewport
- Heading sizes: `!text-[60px] xl:!text-[72px]` for hero headings due to Turbopack `@layer base` issue
